## Project (Work in progress)

Working alongside [AJ](https://github.com/aj-govil) and [Vincent](https://github.com/vincenttiltin) on our [FocusApp](https://github.com/aj-govil/AppDevII-FocusApp)

Whilst it is still a work in progress, it's been equally fun and challenging to pickup Kotlin and Jetpack Compose for the first time to create an app that I actually would want to use

## Vision

# Our goal here was to create a task list manager that emphasizes working on what matters to you most.

You can track our progress on [our public repository here](https://github.com/aj-govil/AppDevII-FocusApp)

To do this we:

- Limit number of tasks to 3 
- Minimal UI, focused on keeping YOU focused on what you need to
- Account preferences for theme so you can pick what is best for you
